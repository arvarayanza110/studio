# **App Name**: Line Commander

## Core Features:

- Color Command Buttons: UI buttons that, when pressed, send commands to the ESP32 microcontroller to follow a specific color line.
- Data Dashboard: Real-time dashboard to visualize data received from the ESP32, such as sensor readings and robot status.
- Data Analytics and Graphing: Display data in interactive graphs, allowing users to analyze the robot's performance over time.
- Communication Bridge: Establish a communication link between the web application and the ESP32, such as websockets.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a tech-focused aesthetic.
- Background color: Very light gray (#F5F5F5) to ensure clarity and readability.
- Accent color: Soft green (#8BC34A) to indicate activity, success, and good connections.
- Body and headline font: 'Inter' sans-serif font for a clean, modern look. 'Source Code Pro' monospaced font to display transmitted or received commands.
- Use icons that are related to robotics, data analysis, and connectivity.
- Dashboard with clear sections for controls, real-time data, and historical graphs.
- Subtle transitions when the robot is changing states based on user activity.
